## JobFinder

Job Finder is a simple LinkedIn-style web application created as part of the SD-105 final project.  

## Features
-The application includes:
- A **login page**
- A **home page** with a **3-column layout**
- Random users displayed using the **Random User API**
As ,The application uses the **Random User API** to fetch user data.

- At least **10 users** are fetched
- Displays:
   Profile picture
   Full name
   City
 Data is retrieved using the **Fetch API**
