## JobFinder

Professional front-end prototype for a job-networking application. This repository contains static HTML, CSS and JavaScript assets intended as a polished UI mockup suitable for inclusion in a professional portfolio or deployment via GitHub Pages.

## Features
- Clean and responsive login page
- Username/email + password input fields
- “Remember me” option
- Social login buttons (Facebook, Google, LinkedIn)
- Home page with:
- Navigation bar
- Profile sidebar
- Post creation box
- Posts feed with like, comment, and share counters
- Suggestions panel
